管理ページについて

G's BBS [XOOPS] は、掲示板の設定などほとんどの管理操作を管理室内からブラウザ上で行うことができます。基本設定、CSS設定、インフォメーション編集、記事の編集、削除、IP アドレスによる書き込み制限、投稿者情報（ USER AGENT、IP ADDRESS など ）の参照などはブラウザを通じて XOOPS管理メニューから管理ページに入り行えます。

● 基本設定

G's BBS [XOOPS] の様々な設定をこのページで行います。スクリプトをサーバへアップし、パーミッションを整えた後にこのページで好みの設定にしてください。各設定項目には詳しい説明が付いていますのでそちらを参考にしてください。

● その他設定

掲示板トップに表示されるインフォメーションの編集と、デザインを指定するスタイルシートの編集をこのページで行います。

● 記事管理

ここではツリー形のリストで記事が表示され、任意の記事タイトルをクリックすると、その記事が含まれるスレッドが開きます。

■記事を編集する

各記事タイトルの前に表示されている[番号]をクリックしてください。編集画面が表示されます。

■投稿者のデータを見る

「DATA」ボタンをクリックすると、投稿時に入力されたすべての値と投稿日時、投稿者の USER AGENT 情報、IP アドレス、REMOTE HOST ( IP アドレスから取得 ) など、投稿者に関して記録されている全ての情報を見ることができます。

■記事の一括削除

管理ページでは、記事をまとめて削除することができます。削除したい記事の左側にあるチェックボックスにチェックを入れて「削除」ボタンを押して下さい。確認画面は出ず、すぐに削除処理が行われますのでご注意下さい。また、返信記事を残したままそのスレッドの親記事を削除した場合、親記事のない返信記事はリストの最上部に別枠で表示されます。これらの返信記事はユーザに対しては非表示となります。記事番号をクリックし、操作を行って下さい。

● IP制限

■ IP ADRESS による書き込み制限

「IP制限」リンクをクリックすると、IP 制限の編集ページが開きます。制限したい IP ADRESS を登録するには、「▼ 制限対象 ▼ 」欄に IP ADRESS を入力して「追加」ボタンを押します。この IP 制限は、IP ADRESS の一部を記述しておくことができます。たとえば、123.456.789.1 という IP のうち、123.456 とだけ登録しておけば IP に 123.456 が含まれるユーザ（ 123.456.123.5 でも 123.456.456.9 でも）は全て書き込みを制限されます。制限を解除したい場合は、解除対象の IP にチェックを入れて「削除」ボタンを押します。確認なしで削除されますのでご注意下さい。制限対象の IP は /modules/gsbbs/ip/exclude.ipd ファイルに格納されます。

■ IP ADRESS による優先的書き込み許可

優先的に書き込みを許可したい IP を登録する場合は、IP 制限編集ページの「▼ 優先許可 ▼ 」欄に IP ADRESS を入力して「追加」ボタンを押します。優先許可に指定した IP は、たとえ制限対象になっていても優先的に書き込みが許可されます。例えば、123.456 が制限対象になっている場合、123.456.789.1 と 123.456.123.1 は両方とも書き込み制限されますが、123.456.7 を優先許可に追加すると 123.456.789.1 は書き込めるようになり、123.456.123.1 は制限されたままになります。優先許可を解除したい場合は、解除対象の IP にチェックを入れて「削除」ボタンを押します。確認なしで削除されますのでご注意下さい。優先許可対象の IP は /modules/gsbbs/ip/allow.ipd ファイルに格納されます。
