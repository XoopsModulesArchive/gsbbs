著作権表示について(重要)

G's BBS [XOOPS] は Web Application Factory で作成された G's BBS v4 を元に、Blues Supporter 運営者である Sting_Band によりXOOPSモジュール化されたスクリプトです。ゆえに G's BBS [XOOPS] も以下に掲げる G's BBS v4 の利用規約を遵守し、使用条件とします。

G's BBS v4 はオープンソースで提供され、以下に述べる制限内であれば無料で使用できるとともにカスタマイズも自由ですが、GPL などの一般的なライセンス形態に完全には準拠せず、独自の利用制限を設けています。以下の制限事項をよくお読みになり、著作権の保護にご協力いただけますようお願い申し上げます。以下、G's BBS v4 そのもの及び G's BBS v4 を基にカスタマイズされたスクリプトをまとめて G's BBS と表記します。

● 著作権の表示

G's BBS は掲示板の最下部に「 G's BBS v4 Beta, Copyright(C) 2003, <a href="http://www.waf.jp/" target="_blank">Web Application Factory</a> All Rights Reserved. 」という HTML タグが表示されます。これは、作者である 久保田英典 ( ハンドルネーム：GUSTAV ) が運営するサイト Web Application Factory へのリンクですが、この著作権表示と Web Application Factory へのリンクが掲示板に表示されていれば完全に無料での使用を許可します。

● カスタマイズ

上記の著作権表示とリンクが表示されていて、G's BBS を基にカスタマイズされたものでるということが明記されていれば、掲示板の色、ページデザイン、スクリプトの変更は自由に行っていただいてかまいません。カスタマイズ版を第三者に配付する場合、できるだけ 作者 までお知らせいただければ嬉しいです。

● 再配布

G's BBS は、Web Application Factory から提供されるアーカイブファイルそのままの状態での無料再配付は制限しません。G's BBS を基にしてカスタマイズされたスクリプトについては、上記の著作権表示とリンクが明示されている形態であれば無料再配付可能です。ただし、第三者に有料でこれらのスクリプトを譲渡することはできません。詳しくは下記の「商用利用の制限」をお読み下さい

● 商用利用の制限

ここでの商用利用とは、G's BBS そのもの、あるいは G's BBS の利用を他者に有償で提供することと定義します。以下に著作権表示をしていても無料での使用を許可しない例を示します。

  • G's BBS を有償で他者に提供する
  • G's BBS の設置を代行して報酬を得る
  • G's BBS のカスタマイズを代行して報酬を得る
  • G's BBS を含めた Web の構築業務を請け負う
  • G's BBS を設置し、有償で他者にレンタルする
  • 会員制等の有料サイトで、有料エリア内に G's BBS を設置する

以上のような形で G's BBS を利用したい場合や、著作権表示を外して利用したい場合、内容に応じて有償としますので 作者 までご相談下さい。

G's BBS を利用することにより結果として収益が上がる場合でも、以下のようなケースでは無料での使用を許可します。ただし、著作権表示を外すことはできません。

  • G's BBS 上に広告を表示して収益を得る
  • 社内 LAN 内での業務利用
  • 雑誌や無料DLサイト等への収録、掲載、添付（アーカイブファイルに限る。要事前連絡）
  • 無料レンタル掲示板スクリプトとしての使用

いずれにしても無料で使用する場合は著作権表示を外すことはできませんので十分ご注意下さい。
